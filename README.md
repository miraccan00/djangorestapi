# Django-Rest-API

## Installation
### create virtualenv after run 'pip install -r requirements.txt' without quote 
then run these codes commandline

``` 
    python manage.py makemigrations
    python manage.py migrate
    python manage.py createsuperuser 
```
create superuser and use it :)

If you want test my API you can import 'Django-Rest-API/API Course Site Tutorial.postman_collection.json' I wrote documentation inside of postman
